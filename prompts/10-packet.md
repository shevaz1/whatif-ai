# Work Packet Template

## 1. 이번 패킷 목표
- 이 패킷에서 달성해야 할 단일 목표를 1문장으로 정의한다.

---

## 2. 완료조건 (Definition of Done)
아래 조건을 모두 만족해야 이 패킷은 완료다.

- [ ] 기능이 실제로 동작한다
- [ ] Loading / Error / Empty 상태가 구현돼 있다
- [ ] WebView 환경에서 문제없는 구조다
- [ ] TDS 컴포넌트를 우선 사용했다
- [ ] Granite 빌드(outdir) 규칙을 깨지 않았다

---

## 3. 관련 제약 / 문서
- Apps in Toss 문서 근거 요약 (@docs)
- 참고한 공식 예제 또는 규칙

---

## 4. 변경 사항
- 수정/추가한 파일 목록
- 핵심 변경 포인트 요약

## 5. 검증 방법

- 실패 시 확인 포인트
